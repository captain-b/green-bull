Predictions

Green Bull is a decentralised application (Dapp) that that allows users to earn crypto by predicting if the future price of a token. Connect your wallet to get started within minutes. No registration required.


GBULL Token

The GBULL token is what feeds the bull. You can trade it for a profit, spend it or vote with it. 30% of the Bull's treasury is allocated each week to buy backs and burns.
